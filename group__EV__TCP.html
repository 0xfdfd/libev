◆ ev_tcp_read()

int ev_tcp_read ( ev_tcp_t * sock,
ev_buf_t * bufs,
size_t nbuf,
ev_tcp_read_cb cb,
void * arg )

Read data.

Once ev_tcp_read() return #EV_SUCCESS, it take the ownership of req, so you should not modify the content of it until bounded callback is called.

It is a guarantee that every bounded callback of req will be called, with following scene:

  • If read success or failure. The callback will be called with read status.
  • If pipe is exiting but there are pending read request. The callback will be called with status EV_ECANCELED.
Parameters
[in]sockSocket handle
[in]bufsBuffer list
[in]nbufBuffer number
[in]cbRead result callback
[in]argUser defined argument.
Returns
ev_errno_t
Examples
tcp_echo_server.c.
