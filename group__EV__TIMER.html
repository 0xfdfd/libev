◆ ev_timer_start()

int ev_timer_start ( ev_timer_t * handle,
uint64_t timeout,
uint64_t repeat,
ev_timer_cb cb,
void * arg )

Start the timer. timeout and repeat are in milliseconds.

If timeout is zero, the callback fires on the next event loop iteration. If repeat is non-zero, the callback fires first after timeout milliseconds and then repeatedly after repeat milliseconds.

Parameters
[in]handleTimer handle
[in]timeoutThe first callback timeout
[in]repeatRepeat timeout
[in]cbActive callback
[in]argUser defined argument.
Returns
