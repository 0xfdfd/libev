ev_random_req_t * req,
void * buf,

ev_random_cb cb )

Fill buf with exactly len cryptographically strong randoms bytes acquired from the system CSPRNG.

Parameters
[in]loopEvent loop. If set to NULL, this function operation in synchronous mode.
[in]reqRandom request. In synchronous mode, it must set to NULL.
[in]bufThe buffer to fill.
[in]lenThe number of bytes to fill.
[in]flagsReserved for future extension and must currently be 0.
[in]cbResult callback. In synchronous mode, it must set to NULL.
Returns
ev_errno_t. When success, short reads are not possible.
