◆ ev_library_shutdown()

void ev_library_shutdown ( void )

Release any global state that holding onto.

Warning
Only call ev_library_shutdown() once.
Don’t call ev_library_shutdown() when there are still event loops or I/O requests active.
Don’t call libev functions after calling ev_library_shutdown().
